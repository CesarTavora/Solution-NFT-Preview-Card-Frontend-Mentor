NFT Preview Card Component

Projeto desenvolvido como parte dos desafios do Frontend Mentor, com foco em aprimorar habilidades em HTML, CSS e responsividade. O objetivo foi criar um componente visual moderno, fiel ao layout proposto, com interações sutis e design limpo.

Visual do Projeto
O layout apresenta um card de visualização de NFT, contendo imagem, título, descrição, valor em Ethereum, tempo restante e dados do criador.

Funcionalidades
• Layout fiel ao design do desafio
• Efeito hover sobre a imagem
• Design responsivo para diferentes tamanhos de tela
• Tipografia e cores padronizadas
• Estrutura semântica com HTML5

Tecnologias Utilizadas
• HTML5
• CSS3
• Flexbox
• Google Fonts

Objetivo do Projeto
Este projeto foi desenvolvido com o objetivo de treinar construção de layouts, organização de código, responsividade e atenção aos detalhes visuais, simulando um componente real de aplicação.
